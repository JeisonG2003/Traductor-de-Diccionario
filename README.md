# **Traductor-de-Diccionario – Semana 11**

Repositorio del proyecto desarrollado para la asignatura Estructura de Datos (E) en la Universidad Estatal Amazónica, carrera de Ingeniería en Tecnologías de la Información.

## **Descripción**

Este proyecto contiene la solución al ejercicio práctico correspondiente a la Semana 11, enfocado en la implementación de mapas y diccionarios utilizando el lenguaje de programación C#.

El sistema consiste en un traductor básico entre inglés y español, utilizando la estructura de datos Dictionary<TKey, TValue> para almacenar asociaciones clave – valor. El programa permite traducir frases completas de manera parcial, es decir, únicamente se traducen las palabras que se encuentran registradas dentro del diccionario.

Además, el sistema permite agregar nuevas palabras dinámicamente durante la ejecución, demostrando el uso de operaciones como inserción de datos, validación de claves con ContainsKey() y acceso eficiente mediante clave.

El proyecto fue desarrollado y ejecutado en Visual Studio Code, mostrando los resultados directamente en consola mediante un programa estructurado bajo el enfoque de Programación Orientada a Objetos (POO).

## **Ejercicios desarrollados**

* Implementación de un diccionario (Dictionary<string, string>) en C#.
* Registro inicial de al menos 10 palabras base en inglés con su traducción al español.
* Desarrollo de un menú interactivo en consola.
* Traducción parcial de frases ingresadas por el usuario.
* Registro dinámico de nuevas palabras dentro del diccionario.
* Validación de existencia de claves utilizando ContainsKey().

## **Objetivos**

* Aplicar la estructura de datos Diccionario (Mapas) en C#.
* Implementar asociaciones clave – valor mediante Dictionary<TKey, TValue>.
* Desarrollar un sistema interactivo utilizando consola.
* Aplicar Programación Orientada a Objetos para organizar el proyecto en carpetas (Modelo, Vista y App).
* Publicar el proyecto en un repositorio público de GitHub como evidencia del versionamiento.
